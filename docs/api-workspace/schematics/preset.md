# preset [hidden]

Create application in an empty workspace

## Usage

```bash
ng generate preset ...

```

## Options

### name

Type: `string`

The name of the application.

### npmScope

Type: `string`

Npm scope for importing libs.

### preset

Type: `string`

The name of the preset.

### style

Default: `css`

Type: `string`

The file extension to be used for style files.
