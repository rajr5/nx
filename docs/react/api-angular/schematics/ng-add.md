# ng-add [hidden]

Add @nrwl/angular to a project

## Usage

```bash
nx generate ng-add ...

```

## Options

### e2eTestRunner

Default: `cypress`

Type: `string`

Test runner to use for end to end (e2e) tests

### skipFormat

Default: `false`

Type: `boolean`

Skip formatting files

### skipInstall

Default: `false`

Type: `boolean`

Skip installing after adding @nrwl/workspace

### unitTestRunner

Default: `jest`

Type: `string`

Test runner to use for unit tests
