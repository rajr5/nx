# workspace-schematic

Generates a custom schematic

## Usage

```bash
ng generate workspace-schematic ...

```

### Options

| Name         | Alias | Description           | Type    | Default value |
| ------------ | ----- | --------------------- | ------- | ------------- |
| `name`       |       | Schematic name        | string  | `undefined`   |
| `skipFormat` |       | Skip formatting files | boolean | `false`       |
